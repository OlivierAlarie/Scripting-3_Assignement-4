# Rock Paper Scissors Battle
The fate of the world is in your hand! Your opponent cannot way to destroy you in this crazy showdown! 1-2 minutes of gameplay.
This game was an assignment for Fourat's class of the Inependent Video Game Design AEC program at Dawson college.

# Controls  
Left-Click on a button - Confirm button
Escape - Open/Close pause menu
